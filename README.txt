Make phone calls using Twilio. 

== Description ==

Make and receive phone calls using the Twilio API, which for most people will be cheaper than using Skype for buying a local number to make calls with. 

== Installation ==

1. Go to Plugins -> Add New -> Upload Plugin in the WordPress admin
2. Upload the zip file which can be downloaded here - https://github.com/jacobschweitzer/wordcall/archive/master.zip
3. Activate WordCall
4. Adjust your settings in the WordCall page under Users -> WordCall
5. Make calls using Twilio
